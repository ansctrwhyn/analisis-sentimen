Aplikasi Analisis Sentimen pada Twitter Mengenai Pelayanan Ekspedisi SiCepat Menggunakan Metode K-Nearest Neighbor

Petunjuk Penggunaan
  1. Import dataset dengan menekan tombol Import Data pada menu Dataset.
  2. Lakukan text preprocessing dengan menekan tombol Preprocessing pada menu Dataset.
  3. Hasil text preprocessing berupa cleansing, case folding, tokenizing, filtering, dan stemming akan ditampilkan pada menu Text Preprocessing.
  4. Lakukan pengujian model klasifikasi menggunakan Cosine Similarity pada sub menu Pengujian 1.
  5. Lakukan pengujian model klasifikasi menggunakan Euclidean Distance pada sub menu Pengujian 2.
  6. Lakukan pengujian uji analisis sentimen dengan melakukan crawling data. Masukkan jumlah data yang akan diambil, lalu tekan tombol Crawling pada sub menu Pengujian 3.
  7. Lakukan uji analisis sentimen dengan menekan tombol Uji Analisis. Kemudian masukkan nilai K (jumlah tetangga terdekat) pada Form Uji Analisis Sentimen, lalu tekan tombol Submit.
  8. Visualisasi hasil uji analisis sentimen akan ditampilkan pada menu Visualisasi Uji Analisis.
